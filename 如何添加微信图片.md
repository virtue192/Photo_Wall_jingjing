# 如何将微信图片添加到照片墙项目

## 步骤1：从微信保存图片到本地

1. 打开微信聊天窗口，找到您想要保存的图片。
2. 右键点击图片，选择"另存为"（或点击图片打开大图后，再右键点击选择"另存为"）。
3. 在弹出的保存对话框中，选择一个您容易找到的位置，例如：
   - 桌面
   - 或系统默认的"图片"文件夹（通常位于`C:\Users\您的用户名\Pictures`）
4. 点击"保存"按钮，将图片保存到本地。

## 步骤2：将图片复制到项目文件夹

1. 打开Windows文件资源管理器（快捷键：`Win + E`）。
2. 导航到您保存微信图片的位置（例如桌面或Pictures文件夹）。
3. 找到您刚保存的图片，右键点击它，选择"复制"（或使用快捷键：`Ctrl + C`）。
4. 导航到Trae项目的images文件夹，路径为：
   ```
   d:\vibe code\写给京京的APP  movies with jingjing\参考的功能GitHub源代码\照片墙\Photo_Wall\images
   ```
5. 在images文件夹内右键点击空白处，选择"粘贴"（或使用快捷键：`Ctrl + V`）。

## 步骤3：处理图片以在照片墙中显示

为了让照片墙能够正确显示并按拍摄日期排序，您需要运行两个Python脚本：

### 步骤3.1：安装所需依赖

首先，确保您的计算机已安装Python和Pillow库。如果没有安装Pillow，可以通过以下命令安装：

```bash
pip install pillow
```

### 步骤3.2：重命名和排序图片

运行`step1_rename.py`脚本，它会：
- 将所有非JPG格式的图片转换为JPG格式
- 读取每张照片的EXIF信息获取拍摄日期
- 按照拍摄日期对照片进行排序并重新命名（从0开始编号）

```bash
python step1_rename.py
```

### 步骤3.3：生成缩略图

运行`step2_thumbs.py`脚本，它会：
- 在`images`文件夹下创建`thumbs`子文件夹
- 为每张照片生成360x360像素的缩略图，以提高页面加载速度

```bash
python step2_thumbs.py
```

## 步骤4：查看效果

完成上述步骤后，您可以直接在浏览器中打开项目根目录下的`index.html`文件，查看照片墙效果。

## 注意事项

1. 如果图片没有EXIF拍摄日期信息，脚本会将它们放在最后并按原文件名排序。
2. 建议使用分辨率适中的图片，过大的图片会影响页面加载速度。
3. 如果您添加了多张图片，可以多次运行脚本进行处理。
